# Data Directory — stages/01-data/data/

## Directory Layout

```
data/
├── bar_data/
│   ├── tick/         ← 1-tick and 250-tick bar exports from SierraChart
│   ├── time/         ← 10-second bar exports from SierraChart
│   └── volume/       ← 250-volume bar exports from SierraChart
├── labels/           ← HMM regime labels (generated by hmm_regime_fitter.py)
└── touches/          ← Zone touch data from ZoneTouchEngine (ZTE) and ray validation
```

## Notes

- **P1 = In-Sample (IS):** Used freely for calibration and search (Pipeline Rule P1).
- **P2 = Out-of-Sample (OOS):** Runs exactly once with frozen params (Pipeline Rule P2).
- **v31 backups** of touch data were moved to `archive/data_snapshots/v31/` on 2026-03-22.
- **Calibration CSV** (`NQ_calibration_V1_1_20260320_calibration.csv`, 130M) moved to `stages/01-data/analysis/calibration_v1_1/` on 2026-03-22.
- All CSV/TXT files in this directory are gitignored. Source of truth is the SierraChart chartbook.
- Period boundaries are defined in `_config/period_config.md`.
