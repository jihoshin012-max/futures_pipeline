# Zone CSV ZB4 Schema
last_reviewed: 2026-03-20
# Required columns for ZB4 zone touch signal files. Validation fails if any column is missing.
# Source ID: zone_csv_zb4 — must match data_registry.md entry.
# File pattern: NQ_ZB4_signals_*.csv (30 columns total, ZB4 format)

| Column | Type | Description |
|--------|------|-------------|
| DateTime | str | Touch event datetime M/D/YYYY H:MM |
| BarIndex | int | Bar index in the source bar series |
| TouchType | str | DEMAND_EDGE or SUPPLY_EDGE |
| ApproachDir | int | Approach direction: 1 (up) or -1 (down) |
| TouchPrice | float | Price at zone touch |
| ZoneTop | float | Zone upper boundary price |
| ZoneBot | float | Zone lower boundary price |
| HasVPRay | int | Whether VP ray is present (0/1) |
| VPRayPrice | float | VP ray price level |
| ZoneWidthTicks | int | Zone width in ticks |
| PenetrationTicks | int | Penetration depth in ticks |
| TouchSequence | int | Touch sequence number for this zone |
| ZoneAgeBars | int | Age of zone in bars |
| ApproachVelocity | float | Approach velocity |
| TrendSlope | float | Trend slope at touch time |
| SourceLabel | str | Source timeframe label (e.g. 15m, 90m, 360m) |
| TFWeightScore | int | Timeframe weight score |
| TFConfluence | int | Timeframe confluence count |
| CascadeState | str | Cascade state (e.g. PRIOR_BROKE) |
| CascadeActive | int | Whether cascade is active (0/1) |
| SessionClass | int | Session class identifier |
| DayOfWeek | int | Day of week (0=Mon ... 6=Sun) |
| ModeAssignment | str | Mode assignment (e.g. M1, M1H, M3, M5) |
| QualityScore | int | Signal quality score |
| ContextScore | int | Context score |
| TotalScore | int | Combined total score (QualityScore + ContextScore) |
| SourceSlot | int | Source slot identifier |
| ConfirmedBar | int | Bar index where signal was confirmed |
| HtfConfirmed | int | Whether higher timeframe confirmed (0/1) |
| Active | int | Whether signal is active (0/1) |

## Notes
- ZB4 format has 30 columns — different schema from ZRA (zone_csv_v2, 33 cols)
- Key differences from ZRA: adds ModeAssignment, QualityScore, ContextScore, TotalScore, CascadeState, TFWeightScore, TFConfluence; drops Reaction, ReactionPeakBar, ZoneBroken, BreakBarIndex, BarsObserved, RxnBar_*/PenBar_* lookahead columns
- Generated by ZB4 running on 250-vol bars — must be paired with bar_data_volume
- Source files use file pattern NQ_ZB4_signals_*.csv per _config/data_registry.md
