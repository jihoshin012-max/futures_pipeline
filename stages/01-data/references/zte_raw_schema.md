# ZTE Raw CSV Schema
last_reviewed: 2026-03-23
# ZoneTouchEngine v4.0 unified zone touch events.
# Source ID: zte_raw — must match data_registry.md entry.
# File pattern: NQ_ZTE_raw_*.csv (52 columns, ZTE format)

| # | Column | Type | Description |
|---|--------|------|-------------|
| 0 | DateTime | str | Touch event datetime YYYY-MM-DD HH:MM:SS |
| 1 | BarIndex | int | Bar index in the 250-vol base bar series |
| 2 | TouchType | str | DEMAND_EDGE, SUPPLY_EDGE, or VP_RAY |
| 3 | ApproachDir | int | -1 (from above, demand) or +1 (from below, supply) |
| 4 | TouchPrice | float | Price at zone touch |
| 5 | ZoneTop | float | Zone upper boundary price |
| 6 | ZoneBot | float | Zone lower boundary price |
| 7 | HasVPRay | int | VP ray present near zone (0/1), filtered by 3x zone width |
| 8 | VPRayPrice | float | VP ray price (0.0 if not near zone) |
| 9 | Reaction | float | Max reaction in ticks (post-touch, from ZRA measurement) |
| 10 | Penetration | float | Max penetration in ticks (post-touch) |
| 11 | ReactionPeakBar | int | Bar index of peak reaction |
| 12 | ZoneBroken | int | Zone broke after touch (0/1) |
| 13 | BreakBarIndex | int | Bar where zone broke (-1 if not broken) |
| 14 | BarsObserved | int | Observation window in bars (-1 if not resolved) |
| 15 | TouchSequence | int | Nth touch of this zone |
| 16 | ZoneAgeBars | int | Bars since zone first appeared |
| 17 | ApproachVelocity | float | 10-bar price change in ticks (absolute) |
| 18 | TrendSlope | float | 50-bar slope in ticks |
| 19 | SourceLabel | str | Source timeframe (15m, 30m, ..., 720m) |
| 20 | SourceChart | int | SC chart number |
| 21 | SourceStudyID | int | V4 study ID on source chart |
| 22-28 | RxnBar_30..360 | int | First bar where reaction >= threshold (-1 = never) |
| 29-32 | PenBar_30..120 | int | First bar where penetration >= threshold (-1 = never) |
| 33 | ZoneWidthTicks | float | Zone width in ticks (ZoneTop - ZoneBot) / tick_size |
| 34 | CascadeState | str | PRIOR_HELD, NO_PRIOR, or PRIOR_BROKE |
| 35 | CascadeActive | int | Cascade active flag (0/1) |
| 36 | TFWeightScore | int | Timeframe quality weight (0-25) |
| 37 | TFConfluence | int | Number of TF charts with zone at this price |
| 38 | SessionClass | int | Session: 0=Open, 1=MidDay, 2=Afternoon, 3=OffHours |
| 39 | DayOfWeek | int | Day of week (0=Sun..6=Sat) |
| 40 | ModeAssignment | str | M1F, M1H, M3, M4, M5, or --- (Skip) |
| 41 | QualityScore | int | Zone quality score (0-65) |
| 42 | ContextScore | int | Context score (0-70) |
| 43 | TotalScore | int | QualityScore + ContextScore |
| 44 | SourceSlot | int | Chart slot index (0-8) |
| 45 | ConfirmedBar | int | Bar index where signal was confirmed |
| 46 | HtfConfirmed | int | Higher timeframe confirmed (0/1) |
| 47 | Active | int | Signal active (0/1) |
| 48 | DemandRayPrice | float | Nearest broken demand zone ray price (0 if none) |
| 49 | SupplyRayPrice | float | Nearest broken supply zone ray price (0 if none) |
| 50 | DemandRayDistTicks | float | Distance to nearest demand ray in ticks |
| 51 | SupplyRayDistTicks | float | Distance to nearest supply ray in ticks |

## Notes
- Replaces both zone_csv_v2 (ZRA, 33 cols) and zone_csv_zb4 (ZB4, 30 cols)
- VP_RAY touches have empty scoring columns (ModeAssignment="---", scores=0)
- Ray columns (48-51) are summary only; full ray data is in ray_context CSV
- Generated by ZoneTouchEngine v4.0 on 250-vol bars — must be paired with bar_data_volume
- Import from SC via: make import-zte
