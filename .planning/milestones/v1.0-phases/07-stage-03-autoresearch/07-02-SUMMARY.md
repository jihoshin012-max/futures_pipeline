- **Committed in:** 297eeb5

**2. [Rule 3 - Blocking] Fixed relative_to() path crash in hypothesis_generator.py**
- **Found during:** Task 1 (12-experiment smoke test)
- **Issue:** hypothesis_generator.py used Path.relative_to() to compute engine path from script location; crashed when working directory did not match expected ancestor
- **Fix:** Replaced relative_to() chain with repo-root-relative path passed as CLI argument from driver; engine resolved from `repo_root / "stages/04-backtest/autoresearch/backtest_engine.py"`
- **Files modified:** stages/03-hypothesis/autoresearch/hypothesis_generator.py
- **Verification:** Engine path resolved correctly on first call; all 12 experiments completed
