<objective>
Build the per-hypothesis parameter sweep harness (run_combination_sweep.py) and execute ~1800 experiments on P1a to identify which hypotheses beat the sized-profile baselines from Phase 02.1 at any parameter value. Produce dimensional_winners.json as input to Plan 02 combination testing.

Purpose: Phase 1 screening ran all hypotheses at default params with a fixed trigger that ignores features, so all showed delta_pf=0. This plan runs each hypothesis over its actual param_grid with profile-specific martingale params to discover which hypotheses genuinely improve on the tuned baselines. Without this step, combination testing in Plan 02 would be combining hypotheses that have never been validated.

Output: run_combination_sweep.py harness, phase4_param_sweep.tsv (all ~1800 rows), dimensional_winners.json (hypotheses that beat baseline on >= 2 bar types with their best params)
</objective>
