  <action>
Create `shared/archetypes/rotational/run_hypothesis_screening.py` (line 1: `# archetype: rotational`), modeled on `run_sweep.py` pattern:

1. **load_baselines()**: Read `baseline_results/sweep_P1a.json`, extract `best_per_source` section. Return dict mapping source_id to baseline metrics.

2. **get_base_config()**: Load `rotational_params.json`, set `period="P1a"`, set `hypothesis.trigger_params.step_dist=6.0` (optimized baseline, NOT 2.0 default). This ensures Dimension C/D/E/F filters are tested against the optimized baseline per Pitfall 4 from research.

3. **run_single_experiment(base_config, hypothesis, source_id, bar_data, instrument_info, baseline_metrics) -> dict**:
   - If hypothesis has `exclude_10sec=True` and source_id contains "10sec": return skip result with classification="N/A_10SEC"
   - Deep-copy base_config via `build_experiment_config(base_config, hypothesis)`
   - Set per-source config (same pattern as run_sweep.py — single source_id in bar_data_primary)
   - Time feature computation separately: `t_feat_start`, call `FeatureComputer(cfg).compute_static_features(bars)` (or let simulator handle it), measure `feature_compute_sec`
   - Time simulation: run `RotationalSimulator(config, bar_data).run()`, measure `simulator_sec`
   - Compute metrics via `compute_cycle_metrics()`
   - Compute `delta_pf = metrics["cycle_pf"] - baseline_metrics[source_id]["cycle_pf"]`
   - Compute `delta_pnl = metrics["total_pnl_ticks"] - baseline_metrics[source_id]["total_pnl_ticks"]`
   - Return result dict with all fields including timing

4. **run_screening(experiments, base_config, bar_data_dict, instrument_info, baselines) -> list[dict]**:
   - Load bar data ONCE per bar type (same pattern as run_sweep.py)
   - Iterate experiments from `get_screening_experiments()`
   - Print progress: "Experiment {i}/{n}: {hypothesis_id} on {source_id}"
   - Collect results list

5. **write_results_tsv(results, output_path)**: Write TSV with columns:
   hypothesis_id, hypothesis_name, dimension, source_id, cycle_pf, delta_pf, total_pnl_ticks, delta_pnl, n_cycles, win_rate, sharpe, max_drawdown_ticks, avg_winner_ticks, avg_loser_ticks, beats_baseline, feature_compute_sec, simulator_sec, total_sec, classification

6. **print_timing_profile(results)**: Summarize feature_compute_sec vs simulator_sec across all experiments. Print: total_feature_time, total_simulator_time, feature_pct_of_total. This addresses the tooling checkpoint: if feature_compute > 50% of wall clock, flag it.

7. **CLI entry point**: `python run_hypothesis_screening.py [--output-dir screening_results/] [--single H1] [--dry-run]`
   - `--single H1`: Run only hypothesis H1 on all 3 bar types (for testing/debugging)
   - `--dry-run`: Print experiment list without executing

8. Create `shared/archetypes/rotational/test_hypothesis_screening.py`:
   - Test load_baselines returns correct structure
   - Test get_base_config has StepDist=6.0
   - Test H37 exclusion on 10sec
   - Test H37 NOT excluded on vol/tick
   - Test run_single_experiment returns all required fields
   - Test TSV column headers match expected schema
   - Use synthetic/small data where possible to keep tests fast
